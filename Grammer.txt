13.	BinaryFunction → SetqFunction | BinaryOperatorFunction
14.	SetqFunction → setq identifier Value
15.	BinaryOperatorFunction → BinaryOperator Value Value
16.	OtherFunction → identifier Parameters
17. UnaryBinaryFunction → UnaryBinaryOperator identifier ExtraValue
18. ExtraValue → Value | ε
19.	UnaryBinaryOperator → increment | decrement
20. BinaryMoreFunction → BinaryMoreOperators Value Value Parameters
21. BinaryMoreOperators → + | - | * | \
22.	BinaryOperator → mod | rem | >= | <= | > | < | = | <>
23.	Parameters → Value Parameters` | ε
24.	Parameters` → Value Parameters` | ε
25.	Value → logicalTrue | logicalFalse | string | Atom | ( Function )
26.	Atom → identifier | number
