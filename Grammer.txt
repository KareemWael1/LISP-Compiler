1.	Program → Lists
2.	Lists → List Lists`
3.	Lists` → List Lists` | ε
4.	List → ( Contents )
5.	Contents → Content Contents`
6.	Contents` → Content Contents` | ε
7.	Content → List | Block | Expression | ε
8.	Block → dotimes ( identifier number ) Lists | when ( Expression ) Lists
9.	Expression → Function | logicalTrue | logicalFalse
10.	Function → read | UnaryFunction | BinaryFunction | OtherFunction
11.	UnaryFunction → UnaryFunctionName Value | UnaryOperator identifier
12.	UnaryFunctionName → write | sin | cos | tan
13.	BinaryFunction → SetqFunction | BinaryOperatorFunction
14.	SetqFunction → setq identifier Value
15.	BinaryOperatorFunction → BinaryOperator Value Value
16.	OtherFunction → identifier Parameters
17.	UnaryOperator → increment | decrement
18.	BinaryOperator → + | - | * | \ | mod | rem | >= | <= | > | < | = | <>
19.	Parameters → Value Parameters` | ε
20.	Parameters` → Value Parameters` | ε
21.	Value → logicalTrue | logicalFalse | string | Atom | ( Function )
22.	Atom → identifier | number
